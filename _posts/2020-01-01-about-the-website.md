Layout Construction
-------------------

```{.yml}
.:
- null: assets/{js,json}/*.{js,json} # doesn't exist, in order to avoid that js and json cannot work
- default: # provide live 2D, sidebar, search
  - post: post/*.html # provide information of the post, toc, review
  - page:
    - 404: 404.html # provide tencent charity link
    - archive: tag.html
    - home: index.html # provide information of the website, sidebar
```
